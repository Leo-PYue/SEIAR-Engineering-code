# SEIAR Model

The implementation for "SEIAR Rumor Spreading Model with Antagonistic States in Hypernetworks"

### Environment

* Tested OS: Windows
* Python >= 3.8

### Requirements
﻿
```
pip install -r requirements.txt
```

### Basic Usage
﻿
```
python SEIARModel.py
```
